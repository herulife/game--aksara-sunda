# Asset Prompts Aksara Sunda

## Tujuan
Prompt ini dipakai untuk membuat aset visual dengan AI image generator atau sebagai brief ke ilustrator.

Gunakan gaya umum:
- children educational game
- warm, friendly, soft lighting
- sundanese cultural atmosphere
- colorful but natural
- clean composition
- suitable for UI overlay

## Catatan Produksi
- Untuk karakter, minta `transparent background`.
- Untuk background, minta area kosong di tengah untuk UI.
- Untuk ikon, prioritaskan bentuk sederhana dan terbaca pada ukuran kecil.
